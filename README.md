# Varanasi Real Estate Optimization: Diaspora Accommodation Feasibility Study

An analytical, data-driven real estate framework evaluating hospitality investments in Varanasi to capture high-yield international diaspora and spiritual tourism markets.

## 💡 Core Business Insights
* **Market Inventory:** Hospitality listings in Varanasi are overwhelmingly dominated by standard Hotels and traditional Houses, showing an underserved market for premium, boutique alternative stays.
* **The Diaspora Infrastructure Gap:** Cross-referencing property descriptions reveals a significant shortage of accommodations explicitly outfitted with combined premium features (elevators/lifts + secure gated infrastructure) necessary to attract premium international travelers.
* **Financial Opportunity:** Properties offering specialized diaspora-targeted amenities command high booking volume stability and a calculated +25% structural rental yield premium over basic accommodations.

## 🛠️ Data Engineering & Toolkit
* **Data Processing:** Python 3.12, Pandas, NumPy via Google Colab.
* **Data Cleansing:** Standardized text string matching, localized currency symbol stripping, and robust fallback market price estimation logic based on property type categories.
* **Visualization Engine:** Tableau Public Web Authoring.
